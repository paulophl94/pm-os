## Por Contexto de PM

| Situacao | Framework Recomendado |
|----------|----------------------|
| Preciso priorizar features | RICE (rule em `.cursor/rules/rice-analysis.mdc`) + Gibson Biddle DHM |
| Preciso validar uma estrategia | Rumelt Strategy Kernel + Devil's Advocate |
| Preciso estimar impacto de uma feature | Impact Estimation Framework |
| Preciso mapear posicao competitiva | SWOT Analysis |
| Preciso apresentar para lideranca | Presentation Best Practices |
| Preciso defender uma decisao | Devil's Advocate (para antecipar objecoes) |
