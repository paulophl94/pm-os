# To-Do's

Personal task management system for the PM OS.

---

## Files

| File | Purpose |
|------|---------|
| `tasks.md` | Today's tasks with checkboxes |
| `backlog.md` | Future tasks and low priority |
| `week-priorities.md` | Top 3 for the week (connects to quarter goals) |
| `quarter-goals.md` | Quarter goals with progress tracking |
| `commitments.md` | Tracked promises (I owe / others owe me) |
| `progress-log.md` | Cumulative history from morning briefings |

## Folders

| Folder | Purpose |
|--------|---------|
| `briefings/` | Morning briefing outputs (one per day) |
| `reviews/` | Daily review outputs |
| `weekly-reviews/` | Weekly synthesis outputs |
| `learnings/` | Preferences and anti-patterns captured over time |

## Priority System

| Priority | When to Use |
|----------|-------------|
| **P1** | Urgent, impacts deliveries, blockers |
| **P2** | Important but not urgent, follow-ups |
| **P3** | Nice to have, goes to backlog.md |

## Rules

- **Maximum 3 P1 per day** (2 if many meetings) — focus is essential
- Be specific: "Review checkout PRD" > "Review document"
- Move to completed when done — history shows progress
- Review progress-log weekly for accumulated context
