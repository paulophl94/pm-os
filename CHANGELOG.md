# Changelog

All notable changes to PM OS will be documented in this file.

## [Unreleased]

### Added
- Meeting Prep skill — proactive preparation for upcoming meetings with person context, open commitments, and talking points
- People Context auto-injection rule — silently loads stakeholder context when names are mentioned
- Lean PRD template — lightweight 6-section PRD for features that don't need the full treatment
- Global context shared structure — strategic pillars (`pillars.md`) and assumptions (`assumptions.md`)
- Webapp / PM Co-Pilot dashboard — React + Express local dashboard for briefings, tasks, commitments, initiatives, and people
- Toolkit improvements system — captures and ranks workspace improvement ideas (`toolkit-improvements.md`)
- Level Up feature — discovers unused PM OS capabilities based on usage history
- `/improve` command — reviews and ranks toolkit improvement backlog
- Example One-Pager document (`documents/_EXAMPLE-ONE-PAGER.md`)
- GitHub issue templates (bug report, feature request, template submission)
- CONTRIBUTING.md guide
- `start.mdc` onboarding wizard with interactive HTML guide
- Strategic assumptions tracking with Current Truths, 6-Month Bets, and Anti-Truths

## [1.0.0] - 2026-03-20

### Added
- 14 document templates: PRD (Full + Lean), One-Pager, User Stories, RICE, Risk Analysis, Lean Canvas, Roadmap, Epic Plan, KPI-Revenue, Status Update, Personas, Positioning, User Journey
- 6 multi-perspective reviewers: Engineer, Executive, User Researcher, Data Analyst, Customer, Customer Success
- 6 strategy frameworks: Rumelt Strategy Kernel, Gibson Biddle DHM, Devil's Advocate, Impact Estimation, SWOT, Presentation Best Practices
- 7 workflow skills: Morning Briefing, Daily Review, Weekly Review, Meeting Notes, Competitor Analysis, Metrics Analyzer, Atlassian Helper
- Planning system: quarter goals, week priorities, daily tasks, commitments tracker
- Staleness detection for tasks (>5 days) and commitments (>3 days)
- Context Loading Protocol for strategy-aligned document generation
- Context Generator wizard for guided setup
- PM coaching protocols: next-step suggestions, career insights, workflow improvement capture
- Session protocol with silent start-of-session checks
- Global context templates: Domain Vision, KPIs, Roadmap, Architecture
- Example PRD and One-Pager for reference
- CONTRIBUTING.md and issue templates
