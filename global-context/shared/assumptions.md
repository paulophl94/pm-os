## How to Use This File

- **Before roadmap decisions:** Read Current Truths and 6-Month Bets to confirm alignment

- **Quarterly review:** Move items between sections, invalidate what changed, add new bets
- **When surprised:** If reality contradicts an assumption, move it to Invalidated and assess downstream impact
