# Strategic One-Pager: [Initiative Name]

## When to Use This Template

Use the Strategic One-Pager as your **first step** before investing in detailed requirements. This document is designed for:

- **Early-stage feature proposals** that need stakeholder buy-in
- **Strategic alignment discussions** before committing to detailed spec work
- **Quick decision-making** on whether to proceed with an initiative
- **Communicating trade-offs** to leadership in a concise format

**Time Investment:** 1-2 hours to complete  
**Audience:** Product leadership, engineering leads, stakeholders  
**Next Step:** Once approved, proceed to the [Requirements Spec](prd-requirements-spec.md) for detailed implementation planning
