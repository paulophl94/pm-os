The Editor

What is Cursor?

Cursor is an AI-native editor — think of it as a text editor with a built-in AI assistant. It reads, creates, and edits files in your workspace. You don't need to know how to code. Just open the chat panel and describe what you need in plain language.

    1

    Chat Panel — Your main interface

    Press ⌘L (Mac) or Ctrl+L (Windows) to open it. This is where you talk to Cadence OS. Type what you need in plain language — no special syntax required.

    2

    Three Modes — Agent, Ask & Plan

    Agent reads, creates, and edits files — the default for daily work. Ask is read-only for analysis without changes. Plan researches first, shows a plan, then executes after your approval — the power move for complex tasks like full PRDs and roadmaps. Switch modes from the picker at the top of the chat panel.

    3

    Rules, Skills & @ Context — The brain

    Rules (.cursor/rules/) are persistent instructions that shape every response — 14 templates, 6 reviewer personas, coaching protocols. Skills (.cursor/skills/) are multi-step workflows like daily briefing and meeting processing. Type @ in the chat to reference specific files, folders, @Web for live search, or @Docs for indexed documentation.

    ⚡ Quick Start
    1. Download Cursor and install it
    2. Clone the repo: git clone https://github.com/paulophl94/cadence-os.git
    3. Open the cadence-os folder in Cursor
    4. Press ⌘L to open the Chat Panel
    5. Type: /start — the wizard will ask you to pick a setup tier (Quick 5 min, Full 15 min, or Power 25 min) and guide you from there

    The Toolkit

    14 document templates, 6 review personas, 8 workflow skills, and 6 strategy frameworks — all context-aware. No commands to memorize — just describe what you need in natural language.

    💡
    Pro Tip: Be Specific
    Good: "Create PRD for self-service password reset on mobile"
    Vague: "Create PRD for authentication"

    The more context you give, the better the output. Include the scope, platform, and user segment whenever possible.
